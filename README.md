# WaveWar

###### Developed with Unreal Engine 5
  - ###### Utilizing C++ and Blueprints.
  - ###### Using Unreal Engine's built-in Gameplay Ability System (GAS).
  - ###### Using Common UI.

###### Goal of the game: Survive the onslaught of opponents in each wave!
